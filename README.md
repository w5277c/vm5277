# vm5277 Embedded Java like toolkit
**Java-like embedded toolkit for Atmel, Microchip, STM8, and other MCUs**
Универсальная платформа для embedded-разработки(Java-подобный синтаксис j8b + RTOS + кросс-платформенные драйверы для 8-битных МК)

## Манифест
Я создаю этот инструментарий, потому что считаю, что можно сделать embedded-разработку удобнее.  
Просто исключив из неё Си, заменив его низкоуровневым программированием на ассемблере и высокоуровневым программированием на ООП-языке со схожим с Java синтаксисом.  
При этом не жертвуя производительностью и контролем.  

Это альтернатива для тех, кому тоже кажется, что можно проще, удобнее и эффективнее — и кто готов попробовать иначе.  
*Да, без Си можно лучше, как Java показала в enterprise-мире, и я постараюсь это доказать своим проектом.*
