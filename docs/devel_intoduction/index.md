# Краткое описание проекта

VM5277 — это универсальный инструментарий для embedded-разработки, который предоставляет мост между миром высокоуровневой ООП-разработки (Java-подобный язык J8B) и нативной компиляции под 8-битные микроконтроллеры (AVR, PIC, STM8).

**Статус проекта:** В общей картине проект находится **в стадии активной альфа-разработки**, но в фокусе AVR (ATmega328) реализовано ядро и полный цикл разработки:
- Компилятор J8B с почти полной поддержкой синтаксиса
- Полноценный ассемблер
- RTOS с системными вызовами
- Удовлетворительная оптимизация кода
- Инструменты сборки (Maven плагины, NetBeans плагин, консольные утилиты)
- Набор рабочих примеров.

Основа платформы завершена — сейчас идёт внутреннее тестирование и доработка деталей.  
Далее начнется работа над:
- **Runtime** — расширение стандартной библиотеки и системных сервисов
- **Многозадачность** — планировщик, класс Thread
- **Драйверы периферии** — UART, I2C, SPI, 1-Wire, ADC, таймеры, ...
- **Поддержка других микроконтроллеров AVR** — ATtiny, другие ATmega
- **Улучшение существующего функционала** — оптимизация компилятора, сообщения об ошибках, расширение синтаксиса J8B, более глубокий семантический анализ, развитие инструментария, доработка LSP-сервера, поддержка других IDE, наконец интеративная отладка и т.д.

Эти процессы будут идти параллельно по мере востребованности.

Такой прогресс позволяет считать, что в рамках AVR проект вышел на финальную стадию альфы и приближается к бета-версии, готовой для оценки и тестирования сообществом.

При этом архитектура тулкита имеет основу для кросс-платформенной экосистемы (поддержка PIC, STM8, ...), эти работы начнутся после MVP версии для AVR.


> **Важное предупреждение:** Проект предоставляется "как есть" (AS-IS) без каких-либо гарантий.  
Возможны ошибки, неполная реализация функций и изменения API в процессе разработки.

> **Лицензия:** Проект распространяется под лицензией **Apache 2.0**.  
Это означает свободу использования, модификации и распространения, в том числе в коммерческих целях, при условии сохранения уведомления об авторских правах и лицензии.

> **Если вы знакомы с Arduino:**  
VM5277 заменяет Arduino IDE и C++ на Java-подобный синтаксис, Maven и консольные инструменты.  
Это даёт больше контроля над компиляцией и поддержку современных IDE, таких как NetBeans.

## Основные компоненты проекта:
* **Компилятор J8B** — трансляция Java-подобного кода в оптимизированный ассемблер
* **Ассемблер** — для каждой платформы с общим синтаксисом (единый лексер)
* **RTOS** — операционная система реального времени для каждой платформы
* **Runtime** — мультиплатформенные библиотеки для прикладной логики
* **Унифицированный HAL** — аппаратные абстракции для мультиплатформенной разработки
* **Инструменты** — плагины IDE, плагины Maven, LSP-сервер и утилиты для обновления прошивки
* **J8B примеры** — примеры кода J8B


**Примечание:** *Все инструкции выполняемые в консоли приведены для дистрибутива* ***GNU Linux (Debian)***  
*При необходимости аналогичные инструкции будут дополнены для **Windows**.*
