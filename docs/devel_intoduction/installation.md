## Методы установки тулкита

Тулкит VM5277 можно установить двумя способами:

### 1. Автоматическая установка через Maven-плагин

Этот способ использует Maven-плагины проекта VM5277, которые автоматически скачивают и настраивают тулкит при первой компиляции.

Если у вас не установлен Maven, его можно скачать и установить с официального сайта: [https://maven.apache.org/install.html](https://maven.apache.org/install.html).  
Также вам потребуется Java Development Kit (JDK) — см. официальную инструкцию Maven.  
Шаблон J8B-проекта можно скачать здесь: [j8b_template.zip](https://vm5277.ru/docs/j8b_template.zip).  
При первой сборке проекта будет выполнена проверка наличия тулкита, и при отсутствии он будет установлен в домашний каталог.

**Для сборки проекта необходимо:**

1. Распаковать архив и перейти в распакованную директорию проекта.
2. Отредактировать `pom.xml`, указав ваши параметры:
   - `<j8b.target>` — платформа:чип
   - `<j8b.targetFreq>` — тактовая частота чипа в МГц
3. Собрать проект стандартной командой:
   ```bash
   mvn compile
	```
### 2. Ручная установка (рекомендуется для детального ознакомления)

Этот способ позволяет напрямую загрузить и настроить тулкит, что полезно для понимания структуры проекта и работы инструментов.

Шаги установки:

*	Скачайте и установите дистрибутив Java JRE (если не установлен) не ниже версии 8 под вашу операционную систему.  
Смотрите документацию по стандартной установке Java в официальных источниках, в большинстве случаев достаточно выполнить в консоли:
	```bash
	sudo apt update
	sudo apt install default-jre
	```
*	Скачайте дистрибутив с официального репозитория по ссылке [vm5277-release-latest.zip](https://vm5277.ru/releases/vm5277-release-latest.zip).  
Контрольные суммы можно скачать по ссылке  [checksums.txt](https://vm5277.ru/releases/checksums.txt), версия текущего релиза находится внутри архива файл VERSION.txt
	```bash
	wget https://vm5277.ru/releases/vm5277-release-latest.zip
	```
*	Распакуйте архив в домашнюю директорию
	```bash
	unzip vm5277-release-latest.zip -d ~/vm5277/
	```
*	Настройте переменные окружения:
	```bash
	cd ~/vm5277
	./scripts/path_export.sh
	source ~/.bashrc
	```
*	Проверьте установку:
	```bash
	j8bc.sh -v
	```

	Должна отобразиться версия компилятора, например: 0.5.0

На этом основная настройка тулкита завершена.
> **Дополнительная информация:** *Есть возможность сборки Java приложений в виде нативных исполняемых файлов под определенную операционную систему. Это даст заметное увеличение скорости работы утилит, но общий размер исполняемых файлов может превысить размер установленной Java.*
